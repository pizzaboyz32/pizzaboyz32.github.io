ไดอารี่ของเบส

ไดอารี่ 03-30-2020

30/03/20

วันนี้เป็นวันแรกที่ได้ใช้ Notebook เครื่องนี้ จะเรียกว่า Notebook ก็ไม่ได้ด้วยซิ เพราะนี้คือ Chromebook ที่ลงระบบปฏิบัติการ GalliumOS ที่ซึ่งเป็น Linux เรียกว่า Hybrid Book ละกัน เท่ดี 555

วันนี้เข้างานเที่ยง เพราะช่วงนี้จะต้องทดสอบระบบการถ่ายทอดสดไปเรื่อยๆ จนถึงวันงานเลยละ เพราะโครงการ Gamers And Givers นี้ละ เลยต้องพยายามหนักหน่อย

วันนี้ตอนเช้าก็ไม่มีไรมาก พักผ่อนเต็มที่ เข้างานเที่ยง รู้สึกว่าเราไม่ได้ทำอะไรเลย พยายามหาไรทำ หาแนวคิดกิจกรรมของการแข่งใน Platform บ้าง แต่ก็นะ นึกอะไรไม่ออกจริงๆ เลย สักพักพัสดุส่ง Notebook มา เลยออกมาเอา notebook ก่อนกลับไป ก็ บ่าย 3 ครึ่งละ และก็ทำงานวนๆ ไป งานตอนนี้เราต้องดูไฟท์ ซึ่งก็ดูแล้วละ ราคาอยู่ที่ 1000 บาทต้นๆ หมดเลย น่าจะเพราะ Covid-19 ที่ทำให้ราคาตั๋วเครื่องบินตกได้ขนาดนี้

ตกดึกมา เป็นช่วง Operation Test ทดสอบ รันคิวการถ่ายทอดสด ซึ่งก็อลมาดมากจริงๆ รอบแรก เลยพักกันซักครู่ มารันรอบสอง รอบสองก็ดีขึ้น แต่ก็ยังไม่เป็นที่พอใจเลยนะ

ทดสอบระบบเสร็จ ก็กลับบ้าน ซื้อขนม นม เนยนิดหน่อยก่อนเข้าบ้าน เพราะงานเลิกดึก และหิวมากๆๆๆ555

วันนี้ก็ยังเป็นวันที่เราไม่ได้ตัดสินใจว่าจะอยู่ต่อไหม จะทำไงดีนะ เกณฑ์ทหารก็เลื่อนไปแล้วด้วย พรุ่งนี้ค่อยว่ากันละกัน

เบสสุดหล่อ

ไดอารี่ 03-31-2020

31/03/20

วันนี้ก็เป็นอีกวันหนึ่งที่เราตื่นสาย เพราะเมื่อวานจัดหนักพอตัวเหมือนกัน งานหนักมากช่วงนี้ ต้องทำ Operation Test ทุกๆ วัน

ตื่นมาก็ไม่มีอะไรมาก เช็คเมล เช็คงาน แล้วก็นอนดู Brooklyn 99 เพลินๆ จนถึงเที่ยง กินข้าว แล้วก็ไปทำงาน

ทำงานวันนี้ เฉื่อยมากกก แบบ ทำงานไป เมอไป พยายามวางแผนหาโครงสร้างคุณค่าของกิจกรรมภายในองค์กร ไปทางไหน ก็ดูเหมือนจะเจอทางตันไปหมด สมองตื้อมากๆ ทำงานไม่ได้ประสิทธิภาพเลยวันนี้ สงสัยเพราะว่าเรานอนมากไปแน่ๆ เลย555

ช่วงเวลาทำงานก็ไม่มีอะไรเป็นพิเศษ พี่ๆ ที่ทำงานก็ทำงานเงียบๆ ไป ไม่ค่อยได้คุยเล่นกันซักเท่าไหร่ ต่างคนก็มีงานที่ต้องทำไป

ในช่วง Operation Test นี้ก็ปวดหัวเหมือนเดิม เรื่องการวาง ซีนสัมภาษณ์ ได้ลองหลายๆ รูปแบบ ระบบการจัดการของเราดูไม่พร้อมเลย กับการทำงานสเกลนี้ ผลลัพธ์การทดสอบก็แสดงให้เห็นแล้ว ฟีดแบคกลับมาก็แย่มากๆ ถ้าให้พูดตรงๆ เลย คืองานนี้ไม่เหมาะสมกับทรัพยากร และความสามารถขององค์กรที่เรามีอยู่เลย ทำอะไรก็ไม่ราบรื่น ไม่เหมาะจริงๆ ไม่ควรรับงานนี้มาทำเลย ควรพิจารณาในเรื่องของการรับงานให้ถี่ถ้วนมากกว่านี้ จะดีมาก แต่ทำไงได้ละ ในเมื่องานมันรับมาครึ่งทางแล้ว ก็ The Show Must Go On จ๊ะ

กว่าจะได้กลับบ้าน ก็นู้นละ 4 ทุ่ม แวะซื้อข้าวสาร กับอกไก่ไว้ทำกินพรุ่งนี้ หวังว่า เราจะสามารถผ่านวิกฤตเหล่านี้ไปได้

เออ แล้วก็เรื่องทำงานต่อ สงสัยคงได้ทำงานต่อจริงๆ นั้นละ เพราะการสอบ IELTS ที่ใช้ในการสมัครโควต้าไปออสเตรเลียดันปิดซะงั้น การออกจากงานเพื่อมาเตรียมสอบ คงไม่มีผลอะไรมากแล้วละ เลือกที่จะทำงานต่อดีกว่า ถึงแม้ว่าจะปวดหัว แต่ก็อย่างน้อยได้ตังใช้ละนะ 555

วันนี้บ่นเยอะมากๆ เพราะว่าปวดหัวจริงๆ นั้นละ การเขียนไดอารี่ เราก็อาจไม่คล่องมาก เพราะเพิ่งวันที่สองเองเนาะ เบส สู้ๆ ละกัน

เบสสุดหล่อ

ไดอารี่ 04-01-2020

01/04/20

วันนี้ก็เป็นวันยุ่งๆ อีกวันนะ เมื่อวานดูซี่รี่ย์หนักไปหน่อย กว่าจะตื่นก็ 9 ครึ่ง กว่าๆ สายใช้ได้เลยละ 555 พอตื่นมา ก็ดูซีรี่ย์ต่อ ช่วงนี้ติดจริงๆ นะนิ Brooklyn 99 ดูไป ทำงานไป ขำไป ชีวิตแฮปปี้ 555 วันนี้ก็เป็นวันแรกที่เราวางแผนว่าจะห่อข้าวไปกิน เพราะจะให้สั่งกินทุกวันก็ไม่ไหวแหะ 55

เวลาเที่ยงก็เข้างานปกติ ไม่สาย ไม่เกิน เริ่มงานกันเป็นปกติ วันนี้สมองแล่นช้าหน่อย แต่อย่างน้อยก็พอได้ไอเดียงานมาบ้าง กับการทำระบบนิเวศการแข่งขึ้นมาใน Esports Platform ที่ดูดีใช้ได้เลยละ แต่ต้องทำวิจัยเพิ่มเติมนิดหนึ่ง ว่าผู้เล่นต้องการแบบที่เราคิดรึเปล่านะ

วันนี้รุ่นพี่ก็ประชุมกันหนักมาก เรื่อง Operation Test ที่เกิดขึ้นเมื่อวาน วางแผนกันใหม่ สำหรับ Operation Test วันนี้ ซึ่งผลลัพธ์ออกมา เราก็ว่าดีขึ้นนะ พี่ออฟอาจหัวร้อนหนักไปหน่อย เพราะระหว่าง Test เนี้ย มีการปรับ Sequence งานไปมา นอกสคริปนู้นนี้นั้น ทำให้การรันคิวสะดุดอยู่บ้าง แต่ทุกอย่างก็ดูดีกว่าเมื่อวานเยอะนะ เราว่า 555

วันนี้เราก็ได้ลองเขียน Webhook ร่วมกับ Messenger API ของ LINE ทำเป็นบอทขึ้นมาด้วยละ เขียนง่ายกว่าที่คิดนะ หลังจากที่เราเข้าใจเกี่ยวกับ NodeJS มาระดับหนึ่งแล้ว นอกจากจะทำการโต้ตอบได้ เราก็สามารถ Push ข้อความไปยัง Userได้แล้วด้วย ตอนนี้ก็มีแผนในหัวแล้วละ ต่อไปจะลองสร้างฐานข้อมูลโดยใช้ MySQL ดู และเขียน Script ใส่ในบอท ให้ดึงข้อมูลจาก MySQL ซึ่งถ้าเราทำได้สำเร็จ บอกเลยว่า สามารถนำไปใช้ประโยชน์ได้หลากหลายเลยละ ตอนเลิกงาน ก็มาศึกษาต่อ พอที่จะเข้าใจในภาษา MySQL บ้างแล้วละ ตอนนี้เหลือแต่ศึกษาการสร้าง User และการ Intergrate ตัว MySQL เข้าไปในสคริป ก็ไม่น่ามีอะไรเพิ่มเติมแล้วละ

เอ้ย เกือบลืมไป ว่าวันนี้โชคดีด้วยนะ เพราะว่าตอนกลับจากที่ทำงาน แวะซื้ออกไก่ ไปเจออกไก่ลดราคาด้วยละ ซื้อมา โลหนึ่งในราคา 25 บาทกว่าๆ ด้วยละ 5555 ถึงแม้ว่าจะใกล้หมดอายุก็แล้วเหอะ แต่เรากินหมดภายใน 2-3 วันนี้แหละ

วันนี้ก็เป็นอีกวันหนึ่งที่ยุ่งๆ แต่ก็ได้เรียนรู้อะไรใหม่ๆ เยอะเลยละ หวังว่าวันพรุ่งนี้ จะเป็นวันที่ดีขึ้นกว่านี้นะ

02/04/20

วันนี้ตื่นเวลาปกติ 8 โมงเกือบ 9 โมง ตื่นมาแบบอึนๆ หน่อยวันนี้ ตอนแรกกะว่าจะเก็บบ้าน แต่รู้สึกขี้เกียจมาก ร่างกายเหมือนยังไม่เข้าที่เข้าทาง เลยกะว่า เลิกงานค่อยมาเก็บละกัน ถึงแม้ว่าช่วงนี้อาจกลับบ้านดึกหน่อย แต่ทำตามใจตัวเองละกัน

วันนี้ช่วงเช้าก็ศึกษา MySQL ต่อเลย จนตอนนี้เราสามารถเชื่อมต่อกับ MySQL ผ่าน NodeJS ได้แล้ว ดึงข้อมูลออกมาได้แล้ว แต่ต้องใช้เวลาศึกษาเพิ่มเติมนิดหนึ่ง ในเรื่องของภาษา MySQL ให้มากกว่านี้

ทำงานวันนี้รู้สึกเชื่องช้ามาก วันนี้ลองเปลี่ยนมาใช้โปรแกรมทำ Slide อย่าง LibreOffice Impress ดู ปรากฏว่าก็ดีใช้ได้เลยนะ มีฟังก์ชั่นดีๆ เยอะเลยละ ถึงแม้ว่าอาจไม่สู้ Canva กับ PowerPoint แต่เรียกได้ว่า ลูกเล่นค่อนข้างเยอะ และอิสระในการทำงานค่อนข้างเยอะเลยละ เลยกะว่าจะลองทำ Business Plan ของ Esports Platform โดยใช้โปรแกรมนี้ดู ว่าจะเป็นยังไง

วันนี้พี่ออฟให้ผมถ่ายคลิป Mock up ด้วย ไอ้เราก็ถ่ายแบบ งงๆ นั้นละ สคริปอ่านอยู๋ 3 วิ Acting ระดับร้อยล้าน 555 ถือว่าเป็นสีสันระหว่างวันทำงานอันแสนน่าเบื่อละเนาะ
